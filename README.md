# Databraid Sheet API


## Usage

Before beginning make sure you have [Docker](https://www.docker.com/) installed and running, then do:

```
npm i
npm run up
```

When finished developing do:

```
npm run down
```

Testing and linting can all be done respectively with:

```
npm test
npm run lint
```
